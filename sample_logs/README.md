# Sample Network Evidence

This directory contains representative network evidence collected during evaluation of the 5G NFV Digital Forensic Readiness Tool (5GDFRT).

The samples are intended to demonstrate the structure and format of the evidence processed by the prototype and are provided for educational and reproducibility purposes.

## Directory Structure

| Directory     | Description                     |
| ------------- | ------------------------------- |
| `benign/`     | Normal network traffic          |
| `portscan/`   | Network reconnaissance activity |
| `ddos/`       | Denial-of-Service activity      |
| `web_attack/` | Web application attack traffic  |

The complete experimental datasets are significantly larger and were generated within the research environment described in the accompanying thesis.
