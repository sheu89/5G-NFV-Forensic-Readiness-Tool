# System Architecture

## Overview

The 5G NFV Digital Forensic Readiness Tool (5GDFRT) is a research prototype developed to demonstrate proactive digital forensic readiness within virtualized 5G environments.

The prototype integrates Network Function Virtualization (NFV), Software Defined Networking (SDN), and centralized log collection to automatically collect, preserve, store, and analyse digital evidence generated by 5G network functions.

## Main Components

- OpenStack DevStack
- OpenStack Tacker
- OpenAirInterface (OAI) 5G Core
- UERANSIM
- VNF Forensic Agents
- Filebeat
- Logstash
- Elasticsearch
- Kibana

## Prototype Workflow

1. Virtual Network Functions are deployed using OpenStack Tacker.
2. 5G network traffic is generated using UERANSIM.
3. Forensic agents continuously capture network traffic.
4. Filebeat forwards forensic logs to Logstash.
5. Logstash normalizes and enriches the collected evidence.
6. Elasticsearch stores indexed forensic artefacts.
7. Kibana provides visualization, anomaly detection, and forensic investigation capabilities.

This repository contains the implementation artefacts supporting the prototype described in the accompanying PhD research.
