# Documentation

## Overview

This directory contains supporting documentation for the research prototype presented in the PhD thesis.

The documentation is organised into several categories.

## Structure

| Folder | Description |
|---------|-------------|
| architecture | System architecture diagrams and design documentation |
| deployment | Deployment procedures and reproducibility guide |
| experiments | Experimental documentation |
| screenshots | Screenshots of the implemented prototype |

## Purpose

The material contained within this directory supports reproducibility of the research and provides supplementary information that complements the thesis.
